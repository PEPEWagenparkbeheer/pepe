Gedeelde opslag koppelen

Volg deze 4 stappen één keer (~5 minuten). Daarna zien alle collega's dezelfde data.

1
Ga naar supabase.com → Start your project → maak gratis account aan.
2
Maak een nieuw project aan. Kies een naam (bijv. autosearch) en een wachtwoord. Wacht ~2 minuten tot het klaar is.
3
Ga in het project naar SQL Editor en plak dit en klik Run:
create table zoekopdrachten (id bigint primary key, klant text, auto text, details text, km text, jaar text, budget text, btw text, email_klant text, wiezoekt text, opmerkingen text, as_email text, kleuren text, opties text, brandstof text, terugkoppeling_txt text, uitgewerkt boolean default false, terugkoppeling boolean default false, dealer boolean default false, inkopen boolean default false, contract boolean default false, akkoord boolean default false, uitgesteld boolean default false);
alter table zoekopdrachten enable row level security;
create policy "open" on zoekopdrachten for all using (true) with check (true);
4
Ga naar Project Settings → API. Kopieer de Project URL en de anon public key en plak ze hieronder.
